
## MADHAV ECOMMERCE SALES DASHBOARD

## Project Overview
Developed an interactive Power BI dashboard for Madhav Store to track and analyze online sales performance across India. The dashboard provides valuable business insights related to sales, profit, customers, categories, and payment methods.


## Objective
To help the store owner to track and analyze their online sales across India.


## Tools Used
- Power BI
- Power Query
- DAX
- CSV Dataset


## Dashboard Highlights
- Sales & Profit Analysis
- State-wise Performance Tracking
- Category & Sub-category Insights
- Monthly Profit Trends
- Customer & Payment Mode Analysis
- Interactive Filters and KPIs


## Key Insights
- Maharashtra recorded the highest sales.
- Clothing category contributed maximum quantity sold.
- COD was the most preferred payment mode.
- Highest profits were observed in January and November.


## Files Included
